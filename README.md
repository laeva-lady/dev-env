# dev-env; my dev setup
- env: keeps track of my dotfiles
- cmds: commands provided to configure/install some stuff
- resources: some misc stuff i need
- bin: some often used script
    - `thor` manages packages
    - `nets` copies configs to the dev-env repo and optionally push to remote
    - `wmake` loops `make`
    - `tmux-sessionizer` taken from ThePrimeagen
    - `vfz` fuzzy find a file and open it in vim
    - `dev-install` manage install the config in the repo to the user's config
